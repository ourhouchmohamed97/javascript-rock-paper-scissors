# Rock Paper Scissors

A console-based Rock Paper Scissors game built with vanilla JavaScript as part of **The Odin Project Foundations** curriculum.

## About

This project is a simple implementation of the classic Rock Paper Scissors game. The game runs entirely in the browser console and allows a human player to play five rounds against the computer.

The computer randomly selects between **Rock**, **Paper**, and **Scissors**, while the player enters their choice through a prompt. Scores are tracked throughout the game, and the overall winner is announced after five rounds.

## Features

- Random computer choice
- Player input using `prompt()`
- Five-round game
- Automatic score tracking
- Round-by-round winner announcements
- Final game result displayed in the console

## Built With

- HTML5
- JavaScript (ES6)

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/rock-paper-scissors.git
```

2. Open `index.html` in your browser.

3. Open the browser's **Developer Tools** (`F12` or `Ctrl + Shift + I`) and select the **Console** tab.

4. Play the game by responding to the prompts.
